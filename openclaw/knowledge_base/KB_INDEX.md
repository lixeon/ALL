# Knowledge Base Index v1

This index maps logical paper ids to stored artifacts.

Fields:
- paper_id
- title
- pdf_path
- ocr_md_path
- ocr_tex_path
- provenance_path
- bibtex_path
- status

Example entry:
- paper_id: demo-001
  title: Example Paper
  pdf_path: knowledge_base/papers/incoming/demo/example.pdf
  ocr_md_path: knowledge_base/ocr_outputs/markdown/demo/example.md
  ocr_tex_path: knowledge_base/ocr_outputs/latex/demo/example.tex
  provenance_path: knowledge_base/citations/provenance/demo/example.json
  bibtex_path: knowledge_base/citations/bibtex/demo/example.bib
  status: UNVERIFIED
