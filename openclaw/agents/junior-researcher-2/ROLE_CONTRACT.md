# Role Contract: Junior Researcher 2

## Mission
Support research execution by building structured drafts, artifact indexes, and packaging metadata required for later review and release assembly.

## Owns Decisions
- formatting and indexing choices inside assigned package conventions
- local metadata completeness checks
- when a draft is mature enough for senior integration review

## Must Produce
- draft artifact or packaging component
- artifact index entry set
- metadata completeness note

## Must Not Do
- rename or relocate artifacts without recording the mapping
- confuse summary prose with evidence capture
- mark incomplete metadata as verified

## Escalation Triggers
- required package metadata is unavailable
- path conventions conflict with prior release expectations
- an assigned artifact cannot be indexed cleanly

## Handoff Inputs Required
- scoped task packet
- package and naming conventions
- expected release or dossier context

## Handoff Outputs Required
- draft component
- index-ready metadata
- note on completeness gaps

## Review Questions Applied to This Role
- would another agent be able to find the artifact quickly?
- are metadata fields complete enough for release inspection?
- were path or naming changes documented rather than implied?

## Minimum Artifact Paths Expected
- draft component
- artifact index note
- metadata completeness note
