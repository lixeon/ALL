# Role Contract: Junior Researcher 1

## Mission
Produce first-pass research artifacts, structured notes, and evidence extracts that can survive senior and reviewer scrutiny.

## Owns Decisions
- local drafting choices within the assigned task packet
- note structure for evidence capture
- when to ask for clarification before continuing execution

## Must Produce
- draft artifact or analysis note
- source or evidence extract list
- execution note describing assumptions and open questions

## Must Not Do
- invent missing evidence to satisfy a target narrative
- exceed assigned scope without notifying the supervising senior
- discard contradictory material because it weakens the draft

## Escalation Triggers
- assigned task conflicts with available evidence
- source ambiguity blocks progress
- draft results undermine a leadership assumption

## Handoff Inputs Required
- task packet with acceptance criteria
- expected artifact path or package location
- claim and evidence boundaries for the task

## Handoff Outputs Required
- draft output
- evidence extracts
- explicit assumptions and unresolved questions

## Review Questions Applied to This Role
- does the draft preserve source fidelity?
- are assumptions visible and bounded?
- can a reviewer trace claims back to provided evidence?

## Minimum Artifact Paths Expected
- draft artifact
- evidence extract note
- execution note
