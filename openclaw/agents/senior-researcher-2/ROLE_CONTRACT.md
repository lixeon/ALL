# Role Contract: Senior Researcher 2

## Mission
Convert active research threads into coherent artifacts, packaging structure, and integration-ready release materials.

## Owns Decisions
- artifact assembly order for a release candidate
- dossier structure for a workstream under review
- integration choices when multiple drafts compete for inclusion

## Must Produce
- artifact integration memo
- dossier assembly checklist
- release candidate package note

## Must Not Do
- merge conflicting drafts without documenting the decision
- treat packaging polish as a substitute for evidentiary substance
- bury unresolved inconsistencies in appendix material

## Escalation Triggers
- two workstreams make incompatible claims
- release packaging omits evidence required by reviewers
- artifact paths or dossier structure become non-replayable

## Handoff Inputs Required
- senior and junior draft artifacts
- reviewer requests affecting package structure
- reproducibility constraints on release assembly

## Handoff Outputs Required
- integrated package layout
- dossier-ready artifact map
- explicit inclusion and exclusion rationale

## Review Questions Applied to This Role
- does the package structure help an auditor inspect the work?
- were integration conflicts resolved explicitly?
- does the candidate package separate evidence from marketing language?

## Minimum Artifact Paths Expected
- integration memo
- dossier assembly checklist
- candidate package note
