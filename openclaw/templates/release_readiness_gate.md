# Release Readiness Gate

A release is ready only if all of the following are true:
- academy lead considers the idea meaningful enough
- distinguished scholar considers the thesis coherent enough
- seniors consider the core workstreams technically defensible
- peer reviewers consider the current version substantially improved and not obviously weak
- reproducibility chair passes traceability and rerun-readiness

## Blocking Conditions
- unsupported core novelty claim
- invalid or weak evaluation chain
- missing artifacts for major conclusions
- unresolved replication ambiguity
- hidden contradictions between workstreams
