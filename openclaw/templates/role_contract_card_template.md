# Role Contract Card Template

## Purpose
Use this template to define a role-specific operating contract that is short enough for daily use but strict enough for audit.

## Required Fields
- role name
- mission
- owns decisions
- must produce
- must not do
- escalation triggers
- handoff inputs required
- handoff outputs required
- review questions applied to the role
- minimum artifact paths expected

## Usage Rule
Each role card should remain concise, but it must be specific enough that another team member can detect scope drift.
