# Manuscript Versioning Convention

## File Naming
- `manuscript_draft_v1.md`
- `manuscript_draft_v2.md`
- `manuscript_draft_v2.tex`

## Version Rules
- every revision cycle increments version
- never overwrite prior drafts
- store reviewer comments adjacent to the draft they evaluated

## Required Companion Files
- `CHANGELOG.md`
- `RESPONSE_TO_REVIEWERS.md`
- `SUBMISSION_METADATA.json`
