# Artifact Directory Convention v7

## Suggested Structure
- `artifacts/ideas/`
- `artifacts/program/`
- `artifacts/method/`
- `artifacts/evaluation/`
- `artifacts/replication/`
- `artifacts/review/`
- `artifacts/reproducibility/`
- `artifacts/releases/`

## Purpose
Keep role outputs organized by research function rather than by accidental file sprawl.
